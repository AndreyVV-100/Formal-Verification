# Курс по формальной верификации

Этот репозиторий содержит некоторые материалы (семинары, дз) по курсу формальной
верификации, который читается в ИСП РАН. Автор не отвечает за корректность
чего-либо (но вы всегда можете попытаться это формально доказать).

## Frama-c

Гайд по установке `frama-c` с плагином AstraVer можно найти [здесь](https://forge.ispras.ru/projects/astraver/wiki).

Запуск:

```bash
frama-c -av file.c
```

Рекомендуется делать двойной split, после чего доказывать с помощью CVC5.
`frama-c` нужен для `Lecture 3`, `Lecture 5`, `Task 1` и `Task 1 v2`.
`Task 1` - неудачная, но более правильная попытка доказательства сортировки Шелла.

## DPLL

Собственный SAT-решатель, использующий формат DPLL, расположен в папке `Task 2`.
Спасибо семинаристу Артёму Коцыняку за предоставленный парсер DIMACS формата.
Собирается всё с помощью команды `make` (перед этим создайте папку `build`),
на вход подаётся путь до файла в DIMACS формате (см. примеры
[здесь](https://www.cs.ubc.ca/~hoos/SATLIB/benchm.html)). Я тестировал
на наборах `uf150-645 / uuf150-645` и `HANOI` (`hanoi4.cnf`).
